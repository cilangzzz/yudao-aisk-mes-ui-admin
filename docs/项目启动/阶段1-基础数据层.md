# 阶段1：基础数据层

## 一、阶段概述

### 1.1 模块清单
| 序号 | 模块名称 | 英文名 | 依赖 | 启动方式 |
|------|---------|--------|------|----------|
| 1 | 产品管理 | product | 无 | 并行 |
| 2 | 车间管理 | workshop | 无 | 并行 |
| 3 | 班次管理 | shift | 无 | 并行 |
| 4 | 产品BOM | bom | product | 串行 |
| 5 | 产线管理 | prdline | workshop | 串行 |
| 6 | 班组管理 | team | prdline | 串行 |

### 1.2 执行顺序
```
并行启动: product, workshop, shift
    │
    ├── product 完成后 → 启动 bom
    ├── workshop 完成后 → 启动 prdline
    └── prdline 完成后 → 启动 team
```

#### 基本信息
| 项目 | 内容 |
|------|------|
| API文档 | `H:\basePlatform\yudao-aisk-mes\docs\api\mes\basic\08-product.md` |
| 前端路径 | `src/views/mes/basic/product/` |
| API路径 | `src/api/mes/basic/product/index.ts` |
| 依赖 | 无 |

#### 文件清单
```
src/api/mes/basic/product/
└── index.ts                    # API函数 + 类型定义

├── index.vue                   # 列表页（搜索+表格+操作）
├── data.ts                     # 表单Schema + 表格列配置

    └── form.vue                # 新增/编辑表单弹窗
```

| 接口 | HTTP方法 | 路径 | 权限标识 |
|------|----------|------|----------|
| 创建产品 | POST | /mes/product/create | mes:product:create |
| 更新产品 | PUT | /mes/product/update | mes:product:update |
| 删除产品 | DELETE | /mes/product/delete | mes:product:delete |
| 获得产品 | GET | /mes/product/get | mes:product:query |
| 获得产品分页 | GET | /mes/product/page | mes:product:query |
| 获得产品列表 | GET | /mes/product/list | mes:product:query |
| 获得产品精简列表 | GET | /mes/product/simple-list | 无需权限 |
| 导出产品Excel | GET | /mes/product/export-excel | mes:product:export |
