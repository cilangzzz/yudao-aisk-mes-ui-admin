### 2.1 生产追溯 (trace)

#### 基本信息
| 项目 | 内容 |
|------|------|
| API文档 | `H:\basePlatform\yudao-aisk-mes\docs\api\mes\06-trace.md` |
| 前端路径 | `src/views/mes/trace/` |
| API路径 | `src/api/mes/trace/index.ts` |
| 依赖 | operation |

#### 文件清单
```
src/api/mes/trace/
└── index.ts

src/views/mes/trace/
├── index.vue                   # 追溯入口页
├── data.ts
└── modules/
    ├── vin-trace.vue           # VIN正向追溯
    ├── part-trace.vue          # 关键件反向追溯
    └── operation-detail.vue    # 工序作业详情（可选）
```

#### 接口清单
| 接口 | HTTP方法 | 路径 | 说明 |
|------|----------|------|------|
| VIN正向追溯 | GET | /mes/trace/vin | 查询车辆生产过程 |
| 关键件反向追溯 | GET | /mes/trace/part | 查询零件使用车辆 |
| 工序作业详情 | GET | /mes/trace/operation | 工序详情查询 |
| 操作员作业记录 | GET | /mes/trace/operator | 操作员记录查询 |

#### 特殊功能
- **VIN正向追溯**：输入VIN码，查询车辆完整生产过程
- **关键件反向追溯**：输入零件序列号，查询使用该零件的车辆
- **追溯时间线**：以时间线形式展示追溯结果
- **树形追溯展示**：以树形结构展示追溯关系

#### 数据结构
```typescript
export namespace TraceApi {
  export interface VinTraceResult {
    vin: string;
    productCode: string;
    productName: string;
    operations: OperationTrace[];
  }

  export interface OperationTrace {
    operationName: string;
    workstationName: string;
    operatorName: string;
    startTime: string;
    endTime: string;
    keyParts: KeyPart[];
  }

  export interface KeyPart {
    partCode: string;
    partName: string;
    serialNo: string;
    bindTime: string;
  }
}
```

#### 基本信息
| 项目 | 内容 |
|------|------|
| API文档 | `H:\basePlatform\yudao-aisk-mes\docs\api\mes\07-mobile.md` |
| 前端路径 | `src/views/mes/mobile/` |

| 依赖 | operation |

#### 文件清单
```
src/api/mes/mobile/
└── index.ts

src/views/mes/mobile/
├── index.vue                   # 移动端入口
├── data.ts
└── modules/
    ├── scan.vue                # 扫码作业页
    ├── key-part.vue            # 关键件绑定页
    ├── exception.vue           # 异常上报页
    └── task-list.vue           # 任务列表（可选）
```

#### 接口清单
| 接口 | HTTP方法 | 路径 | 说明 |
|------|----------|------|------|
| 扫码解析 | POST | /app-api/mes/operation/scan | 扫码识别 |
| 开始作业 | POST | /app-api/mes/operation/start | 开始 |
| 完成作业 | PUT | /app-api/mes/operation/complete | 完成 |
| 绑定关键件 | POST | /app-api/mes/operation/bind-part | 绑定 |
| 异常上报 | POST | /app-api/mes/exception/report | 上报异常 |
| 获取任务列表 | GET | /app-api/mes/task/list | 任务查询 |

#### 特殊功能
- **响应式布局**：适配 375px-414px 移动端屏幕
- **触屏交互优化**：按钮、表单等触屏友好
- **扫码SDK集成**：调用设备摄像头扫码
- **离线缓存策略**：支持弱网环境使用
