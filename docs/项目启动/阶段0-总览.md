## 四、目录结构规划

```
apps/web-tdesign/src/
├── api/mes/                    # MES API 接口
│   ├── basic/                  # 基础数据 API
│   │   ├── product/
│   │   ├── bom/
│   │   ├── workshop/
│   │   ├── prdline/
│   │   ├── team/
│   │   └── shift/
│   ├── config/                 # 配置层 API
│   │   ├── workstation/
│   │   └── routing/
│   ├── workorder/
│   ├── operation/
│   ├── stock/
│   ├── trace/
│   └── mobile/
│
└── views/mes/                  # MES 页面
    ├── basic/                  # 基础数据页面
    ├── config/                 # 配置层页面
    ├── workorder/              # 生产工单
    ├── operation/              # 装配作业
    ├── stock/                  # 物料管理
    ├── trace/                  # 生产追溯
    └── mobile/                 # 移动终端
```
