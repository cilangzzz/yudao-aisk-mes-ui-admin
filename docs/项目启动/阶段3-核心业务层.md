## 二、模块详情

### 2.1 生产工单管理 (workorder)

#### 基本信息
| 项目 | 内容 |
|------|------|
| API文档 | `H:\basePlatform\yudao-aisk-mes\docs\api\mes\01-work-order.md` |
| 前端路径 | `src/views/mes/workorder/` |
| API路径 | `src/api/mes/workorder/index.ts` |
| 依赖 | product + routing + prdline |

#### 文件清单
```
src/api/mes/workorder/
└── index.ts

src/views/mes/workorder/
├── index.vue                   # 工单列表页
├── data.ts                     # 表单Schema
└── modules/
    ├── form.vue                # 新增/编辑工单
    └── detail.vue              # 工单详情（可选）
```

#### 接口清单
| 接口 | HTTP方法 | 路径 | 说明 |
|------|----------|------|------|
| 创建工单 | POST | /mes/work-order/create | 创建 |
| 更新工单 | PUT | /mes/work-order/update | 更新 |
| 删除工单 | DELETE | /mes/work-order/delete | 删除 |
| 获得工单分页 | GET | /mes/work-order/page | 分页查询 |
| 获得工单详情 | GET | /mes/work-order/get | 单条查询 |
| 下发工单 | PUT | /mes/work-order/release | 下发 |
| 开始生产 | PUT | /mes/work-order/start | 开始 |
| 完成工单 | PUT | /mes/work-order/complete | 完成 |
| 关闭工单 | PUT | /mes/work-order/close | 关闭 |

#### 工单状态流转
```
新建(0) → 下发(1) → 生产中(2) → 已完成(3) → 已关闭(4)
```

#### 操作按钮显示规则
| 状态 | 可用操作 |
|------|----------|
| 新建(0) | 编辑、下发、关闭、删除 |
| 下发(1) | 开始、关闭 |
| 生产中(2) | 完成、关闭 |
| 已完成(3) | 无 |
| 已关闭(4) | 无 |

#### 特殊功能
- **状态按钮**：根据工单状态显示不同操作按钮
- **产品选择**：关联产品下拉
- **工艺路线选择**：关联工艺路线下拉
- **产线选择**：关联产线下拉

#### 接口清单

- **作业状态时间线**：展示作业执行过程
- **关键件绑定列表**：显示已绑定的关键件信息
