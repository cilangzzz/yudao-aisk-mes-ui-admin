## 二、模块详情

### 2.1 工作站管理 (workstation)

#### 基本信息
| 项目 | 内容 |
|------|------|
| API文档 | `H:\basePlatform\yudao-aisk-mes\docs\api\mes\03-workstation.md` |
| 前端路径 | `src/views/mes/config/workstation/` |
| API路径 | `src/api/mes/config/workstation/index.ts` |

#### 文件清单
```
src/api/mes/config/workstation/
└── index.ts

src/views/mes/config/workstation/
├── index.vue
├── data.ts
└── modules/
    └── form.vue
```

#### 接口清单
| 接口 | HTTP方法 | 路径 | 说明 |
|------|----------|------|------|
| 创建工作站 | POST | /mes/workstation/create | 创建 |
| 更新工作站 | PUT | /mes/workstation/update | 更新 |
| 删除工作站 | DELETE | /mes/workstation/delete | 删除 |
| 获得工作站分页 | GET | /mes/workstation/page | 分页查询 |
| 获得工作站详情 | GET | /mes/workstation/get | 单条查询 |
| 获得工作站精简列表 | GET | /mes/workstation/list-all-simple | 下拉选项 |
| 按产线查询工作站 | GET | /mes/workstation/list-by-line | 联动查询 |
| 启用工作站 | PUT | /mes/workstation/enable | 启用 |
| 停用工作站 | PUT | /mes/workstation/disable | 停用 |

#### 特殊功能
- 工作站类型：生产工位(0)、检测工位(1)、仓储工位(2)
- 启停控制：需要在表格行中添加状态切换按钮
- 产线关联：表单中需要 `ApiSelect` 关联产线下拉

### 2.2 工艺路线管理 (routing)

#### 基本信息
| 项目 | 内容 |
|------|------|
| API文档 | `H:\basePlatform\yudao-aisk-mes\docs\api\mes\02-routing.md` |
| 前端路径 | `src/views/mes/config/routing/` |
| API路径 | `src/api/mes/config/routing/index.ts` |
| 依赖 | **product** (产品下拉) + **workstation** (工序配置) |

#### 文件清单
```
src/api/mes/config/routing/
└── index.ts

src/views/mes/config/routing/
├── index.vue                   # 路线列表页
├── data.ts                     # 表单Schema
└── modules/
    ├── form.vue                # 新增/编辑路线
    └── process-config.vue      # 工序配置组件（可选）
```

#### 接口清单
| 接口 | HTTP方法 | 路径 | 说明 |
|------|----------|------|------|
| 创建工艺路线 | POST | /mes/routing/create | 创建 |
| 更新工艺路线 | PUT | /mes/routing/update | 更新 |
| 删除工艺路线 | DELETE | /mes/routing/delete | 删除 |
| 获得工艺路线分页 | GET | /mes/routing/page | 分页查询 |
| 获得工艺路线详情 | GET | /mes/routing/get | 含工序列表 |
| 生效工艺路线 | PUT | /mes/routing/activate | 生效 |
| 失效工艺路线 | PUT | /mes/routing/deactivate | 失效 |
| 复制工艺路线 | POST | /mes/routing/copy | 复制 |
| 获得精简列表 | GET | /mes/routing/simple-list | 下拉选项 |

#### 特殊功能
- **嵌套数据结构**：Routing → Operation[] → Material[]
- **工序配置**：工艺路线包含多道工序，每道工序关联工作站
- **版本管理**：生效/失效状态控制
- **复制功能**：支持复制现有工艺路线

#### 数据结构
```typescript
export namespace RoutingApi {
  export interface Routing {
    id?: number;
    routingCode?: string;      // 路线编码
    routingName?: string;      // 路线名称
    productId?: number;        // 产品ID
    productCode?: string;      // 产品编码
    status?: number;           // 状态 0=失效 1=生效
    operations?: Operation[];  // 工序列表
  }

  export interface Operation {
    id?: number;
    operationCode?: string;    // 工序编码
    operationName?: string;    // 工序名称
    workstationId?: number;    // 工作站ID
    sortOrder?: number;        // 排序
    materials?: Material[];    // 物料列表
  }
}
```
